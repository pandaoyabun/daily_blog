WordPressを検討してみようか

May 13, 2009

Movable Type などと同じブログツールの WordPress は、海外でのシェアをかなり伸ばしています。
Movable Type を使っているのは日本が中心となっている状況です。

商用利用が中心となった Movable Type では、まともなデザインのテンプレートが公開されなくなっています。
また、使用言語が Perl か PHP かという違いがありますが、どちらも未経験のスクリプト言語なので、これからどっちを勉強してもそれほど違いのあることでもありません。
今なら有志のユーザーが多くなっている PHP（WordPress）に利があるともいえます。

ここらで、WordPress を導入してみるのも良いのかと思っています。
今日にでも、WordPress をインストールしてみます。

自分で１からデザインするのも楽しいのですが、Movable Type はテンプレートを作るのが大変です。
4.2以降で細かくモジュール化されて、ようやくわかりやすくなってきましたが、昔のバージョンでは、あまり細かいデザインをするとバージョンアップしたときにデザインが崩れることが多かったので、基本は無料のテンプレートを使わせてもらうようにしてました。

Movable Type を商売としてあつかう機会も増えてきたので、今度は WordPress を勉強してみます。
