Windows XP Professional x64

Mar 07, 2006
1 minute read

64bitCPUを購入したのだからと、勢いに乗ってWindows XP Professional x64を購入しました。
ユーザーインターフェイスとかその辺りは、Windows XP Professionalとさほど違いがありませんが、OS内部は全く別物です。
さて、インストールは無事に終了したものの、起動しません。
Windows XPの起動ロゴが表示されたあと、左上でカーソルが点滅する画面で、ドライバのロードが開始されますが、USB周辺をロードしたときに、再起動が発生してしまいます。
何度、チャレンジしても同じところで再起動です。

セーフモードで起動するので、マザーボードに搭載されたチップとかはすべて添付の64bit用ドライバをインストールしてやります。
しかし、解決できません。
自作の鉄則、周辺機器を外して起動すればいいのですが、面倒くさい。

途中で飽きて、デュアルブートにしたWindows XP Professionalを起動して利用しています。

まだしばらくは64bit環境に移行しないですね。
ドライバもそうですが、一般ソフトウェアが64bit用に提供されていないので、32bitエミュレータ動作となり、メリットがないです。

ユーザーを引きつけるようなインターフェイスや独自の機能が盛り込まれていないので、なかなか移行を促すようにはならないのでしょう。
しかし、Windows Vistaを見れば開発環境として提供されている考えるのが妥当であり、Vistaのリリースにはドライバやソフトが充実しているものと思ってます。（期待しています。）

もう少しすれば64bit環境が自宅でも利用できます。
すごい時代です。
