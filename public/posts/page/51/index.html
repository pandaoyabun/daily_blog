It rained not snow

夜半に雪が降っていたのですが、日付が変わる頃には雨に変わってました。おかげで路面が凍ることなく出勤できるのですが、でも凍結していれば、それを口実に休もうかと思ったんですけど。

なぜか先週末より、夕方になると猛烈に眠くなります。うつらうつらと言う感じではなく、眠さに耐えられそうにないので、眠さをかき消すのに色々してみますが、効果なしです。何とか、業務時間の終わりまで耐えて、一眠りします。昼寝どころではなく完全に爆睡です。

体内時計が完全に昼夜逆転しているのだと思います。またそれに加えて残業が続いているので、神経が休む暇がないようです。また、ストレス解消に趣味に使う時間を必ずとるので、必然的に寝不足になります。

なので、雪が降れば休んで、神経を休めようかと思ったのですが。

It snowed after a long absence...

日曜日の未明から降り続いた雪が、日曜日の晩までに２０ｃｍを超え積もりました。

天気予報をチェックしていなかったので、ちょっと驚きました。土曜日の夜に新年会があって家に帰るのが遅かったのですが、そのときは、雪が降ってもいませんでしたので、翌朝起きたときには、ちょっとショックでした。

日曜日に買い物に出ようかと思っていたのですが、それも取りやめて家でごろごろしてましたが、あっという間に一日が終わってしまいました。撮り貯めたアニメ番組をみて、ゲームをちょこっとして、昼寝して。

岡山市内でも積雪があり、岡山南部の山々が白い装いをしているのを始めてみました。さすがに通勤中で写真を撮れなかったのですけれど。

久々に雪景色を撮影せねば。

Cannot I but install Vista?

やはりLANの調子がよろしくない。
またアプリケーションも動きがよろしくない。

ドライバもアプリケーションも、そしてなによりWindows VISTAのシステムファイルをクリーンな状態に戻すため、クリーンインストールせざるを得ないかな。

幸いにも、システムがインストールしてあるドライブで、設定やデータでバックアップしなければならないのはメールとIEのお気に入りとSonicStageCPぐらいなので、それぞれバックアップ方法も簡単なので、思い切ってインストールをしてみようかと思います。

ついでに、ごちゃごちゃしているドライブ構成も整理してみようかと思います。

再インストールに掛かるであろう時間は1時間ほど。今夜にでも実行して何事もなかったように安定させられれば良いのですが。

Self Build Up PC 2008 #03

GeForce8600GTの成果は結構あるみたいです。
Windows VISTAでは、PCの性能を数値で見せてくれる機能があるのですが、それで見ると5.7という数値です。
前のGeForce6600GTx2(SLI)では、4.8ぐらいの数値だったので、かなりの性能向上です。

また、GeForce6600GTの頃は、ファンレス仕様だったのでSLIでは、熱の問題があり、後付で12cmの大型ファンを取り付けていました。
結構低い音で、意外に部屋の外で気になる音でした。

今度のGeForce8600GTは、ビデオカードとしては、比較的大型のファンが付けられています。
色々とファンを取り外すと、総量としての音圧が下がったので比較的静かになりました。
また、周波数も若干上に寄ったので、部屋の外では気にならないレベルになりました。


あと、500GBのHDDですが、さすがにSATA2です。かなり高速です。
前が160GBのATA133にSATA変換を使っていたので、体感差は明らかです。
発熱量も気にならないレベルです。
