すでにWindows Vista Ultimate x64 にしてから、１週間が経ちましたが、世間で言われるほどトラブルに遭遇してはいません。
ソフトウェアも大部分を切り捨てたので、別段、諦めさえあれば、それで快適に使えるものです。

サウンドカードは、ドライバーがメーカーからx64用がリリースされていないので、しばらく待たなければなりませんが、Microsoftから配布されるWDMドライバーで、とりあえずは動いています。
カードは、ONKYO SE-90PCI (VIA Envy24MT を搭載、24bit/192kHz対応)なのですが、16bit/48Khzまでの設定しかできません。
また、QSoundに対応しているようですが、それも享受できません。
しばらく待ちの状態です。

Color Management

Windows Vista から、色管理が簡単になるとのことで、少し調べてみましたが、あまり情報が公開されておらず、概論的なことだけしかつかめません。

色管理はコントロールパネルにあるのですが、どう設定すべきなのかヘルプもなくわかりません。

手持ちの液晶ディスプレイには、ICCプロファイルがついているのでとりあえずインストールしておきました。
ディスプレイの色温度設定は、sRGBにしています。
一応、1677万色表示が可能なパネルではありますが、sRGBの範囲内の1677万色でしょうかね。
また、黒の再現性に乏しいので、厳密な色あわせには使えません。
まあ、それほど厳密さを求めるほどのことに使ってないのでいいのですが。

日本工業規格(JIS)での色調整は、5500Kの標準光下で行わなければならないそうですが、じゃあ、ディスプレイも5500Kにしなくてはいけないのかな。
sRGBに設定したディスプレイは、7200Kよりも若干黄色っぽく、6500Kよりも若干青っぽいです。
室内の蛍光灯は、TOSHIBA ネオスリムZ 3波長タイプ昼光色なので6700Kです。
と、いうことは印刷物を確認するのは、6500K～6700Kのディスプレイ設定と環境光で確認する必要があります。
ちょっと前まで、その調整をしていたのですが、Vistaでは、どの程度調整したらいいのでしょうか。

今までの環境をずばっと切り捨てました。

x64環境では、ハードウェアのドライバ周りがそろっていないので、いろいろと制限がありますが、そこは買い換えの理由にしてしまおうと思っています。

このうち、Vista x64に対応していないのが、SonicStage CP、B's Recorder GOLD7ですか。
まだ、NextFTPを試していません。

PS3

PLAYSTATION3を衝動買いしました。

スキャナーを買うつもりでベスト電器へ行ったんですが、目的のスキャナーが高くて、ほかの店にしようと諦めて、ゲーム売り場に行くと、PS3がいくつか積まれているではありませんか。

PS3(60GBHDD)を購入しました。

かなり高い買い物でしたが、まあ良いでしょう。
これでしばらくは寂しさを紛らわすことができるはずです。
